# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

コーディング面接の練習用リポジトリ。アルゴリズムや数学的問題の実装を Ruby で行う。

## コマンド

### Ruby ファイルの実行

```bash
ruby <filename>.rb
```

### 例: pythagorean.rb の実行

```bash
ruby pythagorean.rb
```

## 言語・環境

- Ruby（標準ライブラリのみ使用）
- ruby-lsp による開発サポート
